01

Hesabgar — Accounting Reconciliation Engine

Python PostgreSQL Excel Automation

Problem

Accounting teams spent hours manually reconciling transactions across multiple data sources, producing reports by hand in Excel. Error-prone, slow, and unscalable.

Solution

Built a one-click reconciliation engine that ingests transaction data, cross-references accounts, identifies discrepancies, and generates formatted Excel reports automatically.

Impact

Reduced reconciliation time from hours to seconds. Eliminated manual errors. Teams now run end-of-day reconciliation with a single command.

02

Tekrari Yab — Duplicate Transaction Detector

Python PostgreSQL Data Validation

Problem

Duplicate entries across financial systems created reconciliation nightmares and inflated reporting. Manual detection was unreliable at scale.

Solution

Engineered an automated duplicate detection system that scans transaction databases, flags duplicates using configurable matching rules, and generates exception reports.

Impact

Catches duplicates that manual review missed. Directly contributed to preventing 100,000+ KWD in financial exposure from undetected duplicate transactions.

03

Mande Yab — Balance Tracking System

Python PostgreSQL Automated Reporting

Problem

Tracking outstanding balances across accounts required pulling data from multiple systems and manual cross-referencing — a time sink prone to oversight.

Solution

Built an automated balance finder that queries databases, computes outstanding amounts, and delivers actionable reports on demand.

Impact

What took a team member half a day now runs in seconds. Improved visibility into outstanding balances across the organization.

04

Remittance Tracking Automation

Python PostgreSQL Performance Optimization

Problem

Tracking remittance status involved 20 minutes of manual lookups per query. With dozens of queries per day, the team was drowning in repetitive work.

Solution

Designed an optimized query system with indexed database lookups and automated status resolution. One command returns complete remittance status instantly.

Impact

20 minutes → 0.05 seconds. A 24,000x speed improvement. The team reclaimed hours every single day.
